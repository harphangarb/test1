🛠️ Step-by-Step Page Layout Specifications
Page 1: Welcome & Disclaimer
Header: Hidden on this page.

Main Container: Centered text layout.

Content Text: Clear safety notice warning visitors to stay within designated visitor paths, refrain from touching unrestored parts, and watch for uneven hangar flooring.

Action Button: A large, centered button filling 80% of screen width on mobile: [ Start Tour ]

Page 2: History of Floyd Bennett Field
Header: Appears at the top. Shows Naval Air Station New York Tour on the left, Page 2 of 20 on the right.

Main Container: Clean blocks of text explaining the transition from New York's early commercial airfield to its critical role as a WWII Naval Air Station.

Navigation Footer: * Left: [ ← Previous Page ] (Subtle outline style)

Right: [ Next Page → ] (Solid blue style)

Pages 3 to 18: Standardized Plane Pages
Every single one of these 16 pages renders the exact same structural template:

Top Section (Image): A responsive image component loading the specific plane image (src="images/plane-id.jpg").

Specifications Grid (4 Columns):

Box 1: Bureau Number (BuNo)

Box 2: Role

Box 3: Top Speed

Box 4: Wingspan

Middle Section (Two-Column Layout):

Left Column (60% width): Model Legacy Paragraph followed by Our Airframe's Service History Paragraph.

Right Column (40% width): A visual box container with a distinct background shade holding the Volunteer Restoration Fact.

Bottom Navigation Bar (Fixed to bottom of view):

Left: [ ← Previous Plane ] (Subtle gray text link/button)

Right: [ Next Plane → ] (Large, high-visibility --accent-orange background button)

Page 19: The Models
Header: Shows Page 19 of 20.

Main Container: Text description detailing the curated display models constructed by H.A.R.P. volunteers.

Navigation Footer: Standard [ ← Previous Plane ] and [ Next Page → ] buttons.

Page 20: Thank You
Header: Shows Page 20 of 20.

Main Container: Closing text thanking visitors for touring Hangar B, along with information on donation stations or how to volunteer with the project.

Navigation Footer: * Left: [ ← Previous Page ]

Right: A distinct button reading [ End & Reset Tour ] which safely sets the page state back to Page 1.
